# SQL2FA

- An application that facilitates a peer-reviewed change management system for implementation analysts update production databases.

## Roles:
- An operator can take one of 3 roles: a requestor, an approver, and a manager
### Requestor
- originates and executes the request

### Approver
- Reviews and approvers/deny the request

### Manager
- Audits requests and runs admin DMLs when needed

## Notes
- Account cannot be created in the application. Contact the DB admin
